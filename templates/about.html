About Me

This is what I do.

This web app is a blogging platform that allows you to create and edit your own posts with an advanced text editor (CKeditor).

You can also customize each page with a different header. The app is built with Python Flask, SQLAlchemy, WTFforms, HTML, CSS, Bootstrap, JavaScript, JQuery, and Ajax. The app also has authentication and hash encrypted secure password for the admin and the users. Users can sign up and comment on the posts, and they get a random profile picture.
